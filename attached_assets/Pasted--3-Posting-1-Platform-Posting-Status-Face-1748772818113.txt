## 3. 게시(Posting) 모니터링 섹션

1. **플랫폼별 게시 현황(Platform Posting Status)**
    - **카드 형태**로 각 플랫폼(Facebook, TikTok, Instagram, Threads, Twitter)별
        - **오늘 게시 건수**
        - **성공 건수 / 실패 건수**
        - **실패율(%)**
        - **마지막 오류 메시지 요약(있을 경우)**
    - **카드 클릭 시 팝업**:
        - 최근 5건의 게시 내역(ID, 콘텐츠 ID, 상태, 게시 시간, 에러 메시지(실패 시) 등 상세 표
2. **실시간 로그 실시간 업데이트(Live Logs)**
    - **스크롤 가능한 콘솔 뷰**로 실시간으로 전송되는 “AI 호출 로그”, “플랫폼 API 호출 로그”, “오류 로그” 등을 보여줌
    - **필터링**:
        - “로그 레벨(INFO, WARN, ERROR)”, “플랫폼(Facebook, TikTok 등)”, “워커 ID”별로 필터링
3. **예약 게시 관리(Scheduled Posts)**
    - **테이블**:
        - 콘텐츠 ID, 스케줄 일시, 대상 플랫폼, 자동화 모드, 승인 상태, 예약 상태(Active/Paused)
        - “예약 편집” 버튼(일시 변경), “예약 취소” 버튼 제공

## 4. 성과(Insights) 분석 섹션

1. **기본 성과 KPI 카드**
    - **총 도달(Reach) / 조회수(Views)**:
        - 지난 7일, 지난 30일 기준으로 플랫폼별 총합
    - **총 반응(Engagement)**: 좋아요, 댓글, 공유, 저장 등 합산(플랫폼별 주요 지표 기준)
    - **평균 참여율(주당/월당 평균)**: “Engagement ÷ Reach” 백분율
2. **플랫폼별 세부 성과 표(Platform Performance Table)**
    - **컬럼**:
        1. 콘텐츠 ID
        2. 게시 일시
        3. 플랫폼(Facebook, Instagram, TikTok, Twitter)
        4. Reach (도달)
        5. Engagement (좋아요+댓글+공유 등)
        6. Views (영상 조회수)
        7. 클릭(CTR, 클릭 수 또는 클릭률)
        8. ROI 지표(광고비/매출 연동 시)
    - **필터/검색**:
        - 기간(오늘, 7일, 30일, 맞춤 날짜 구간)
        - 플랫폼별, A/B 테스트 여부별, 캠페인별 필터 가능
    - **열 숨기기/표시 제어**: 사용자가 원하는 지표만 보여주도록 토글
3. **시계열 차트(Trends Over Time)**
    - **Line Chart 선택 옵션**(탭 전환 또는 드롭다운)
        - “일별 도달 추이”
        - “주별 조회수 추이”
        - “월별 Engagement 추이”
    - **드래그 가능한 날짜 범위 선택**(Zoom In/Out)
    - **툴팁**: 마우스 오버 시 각 날짜 포인트의 세부 지표 나타남
4. **콘텐츠 유형별 비교(Pie/Bar Chart)**
    - **파이 차트**:
        - “이미지 vs 영상” 기여도(총 조회수, 총 Engagement)
    - **막대 그래프**:
        - “플랫폼별 평균 Engagement” 비교
        - “A/B 테스트 승자 vs 비승자 평균 성과” 비교
5. **A/B 테스트 전용 성과 대시보드**
    - **테스트 그룹 리스트**: 간단한 표(그룹 ID, 생성 일시, 변형 수, 승자 상태)
    - **개별 그룹 클릭 시 상세 화면**:
        - **테이블**: 변형별 “플랫폼별 Reach/Views/Engagement” 수치
        - **비교 차트**:
            - “바 차트”로 변형 A vs B vs C의 조회수 비교
            - “라인 차트”로 시간 경과에 따른 성과 추이(예: 게시 후 24시간 간 조회수 증가 곡선)
        - **승자 표시**: 우승 변형 옆에 “🏆” 아이콘 및 “승자!” 태그 표시
        - **다음 액션**: “승자 게시물 청중 재타겟팅” 또는 “비승자 삭제” 버튼
